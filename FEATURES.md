# Feishu ACP Agent - 功能特性

## 🆕 新增功能："正在输入"效果

当用户 @机器人发送消息时，会自动触发以下交互效果：

### 1. 收到消息时
- 在消息上添加 🤔 表情回复（表示"思考中"）
- 设置"正在输入"状态（聊天界面显示"对方正在输入..."）

### 2. 处理完成时
- 自动移除 🤔 表情回复
- 清除"正在输入"状态

```
用户发送消息
    │
    ▼
┌─────────────────────────┐
│ 添加 🤔 表情回复        │
│ 设置"正在输入"状态      │
└─────────────────────────┘
    │
    ▼
┌─────────────────────────┐
│ 消息进入队列            │
│ 等待处理                │
└─────────────────────────┘
    │
    ▼
┌─────────────────────────┐
│ 转发给 Kimi ACP         │
│ 等待响应                │
└─────────────────────────┘
    │
    ▼

│ 发送给用户              │
└─────────────────────────┘
    │
    ▼

│ 移除 🤔 表情回复        │
│ 清除"正在输入"状态      │
└─────────────────────────┘
```

## 📋 所需权限

### 新增权限
- `im:message.reactions:write` - 添加/删除消息表情回复

### 完整权限列表
| 权限 | 说明 |
|------|------|
| `im:message` | 接收消息 |
| `im:message:send_as_bot` | 发送消息 |
| `im:message.reactions:write` | 表情回复（🤔"正在输入"效果） |
| `im:chat:readonly` | 读取群聊信息 |

## 🚀 启动方式

```bash
cd /Users/kirigaya/project/acp-man

```

## 📝 测试方法

1. 在飞书中 @机器人 发送任意消息
2. 观察消息下方是否出现 🤔 表情
3. 观察聊天界面是否显示"对方正在输入..."
4. 等待 Kimi 回复
5. 确认 🤔 表情自动消失

## ⚠️ 注意事项

1. **表情类型**：使用 🤔 (THINKING) 表示"正在思考/处理中"
2. **自动清理**：程序退出时会自动清理所有未完成的表情回复
3. **权限要求**：必须添加 `im:message.reactions:write` 权限
4. **网络要求**：需要能够访问飞书 API (open.feishu.cn)
